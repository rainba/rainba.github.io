markdown编辑器安装与使用


  听人说使用Markdown能够写文章最方便，也看到好多人鄙视MS word无数次，对于我这样的小白而言，多年使用word的经验，倒是没有觉得word有多么难用，而且功能多样。但是听那么多高人推荐Markdown编辑器，于是也下载下来尝尝鲜。目前觉得它配合github甚是好用，直接在markdown编辑器里编辑好的文章更新到github站点上即可，由于对html语言还不能灵活运用，尚不能发挥其完全作用。下面是小白一步一步成长的过程：
一、安装
（1）我安装的是Typora，在网上搜到好多编辑器，这个的特征就是干净小巧，于是毫不犹豫，自己选择适合自己的就好，不必强求；
（2）去官网下载对应版本安装（此处与装QQ类似）；

二、使用
（1）创建文章：
 方法一：点开Typora，保存至Hexo\source_posts文件夹；
 方法二：在hexo文件夹上右键选择git bash here，然后输入hexo new “标题名”，hexo会根据Hexo\scaffolds里面的模板创建一个新的md文件保存在Hexo\source_posts文件夹；当然这里的模板可以修改；

（2）文章头部设置：每一篇博客文章的开头都会有对文章的说明文字，叫做文章头部，文章的头部除了可以设置文章标题、发布日期等基础信息外，还可以对文章添加标签、分类等。
 当我们使用命令 hexo g 生成网站时，Hexo 会自动新建category和tags，出现在hexo/public/categories和tags文件夹中。

（3）首页显示：网站首页显示文章的内容，即在首页只显示文章精简部分，标志前的内容全部显示

（16）图片
    在hexo中插入图片需按一下步骤：
1.在站点配置文件中，将post_asset_folde设置为true；
2.在站点文件夹中打开git bash，输入命令npm install hexo-asset-image –save安装插件；
3.使用hexo new title创建文章，同时在source/_post文件夹中会生成一个与title同名的文件夹，将需要添加的图片放进该文件夹，再通过markdown语法引用即可。
如：![图片文字](文章title/example.jpg"图片描述")
